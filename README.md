# Sample Microservice Boilerplate

A reusable Node.js + Express + TypeScript microservice architecture using Prisma, Joi, Docker, and MySQL.

## Features

- Type-safe codebase with TypeScript
- Prisma ORM with MySQL
- Joi validation with middleware
- Layered architecture (Controller, Service, Repo)
- Centralized response format
- Docker & Docker Compose ready
- Swagger documentation

## Quick Start

```bash
make install
make dev
```

## Docker

```bash
make build
```

Visit Swagger at [http://localhost:3000/api-docs](http://localhost:3000/api-docs)
